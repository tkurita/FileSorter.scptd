set_delegate

set_delegate(a_script)

delegate を設定します。

delegate となる script object は、resolve_container, target_items_at, is_rowwise_for_iconview の何れか、もしくはすべてを実装することによる、FileSorter の動作をカスタマイズできます。

Parameters
  • a_script (script object) : インスタンスの delegate に設定する script object
Result

script object : me
